# syntaqlite — Claude Code Plugin

SQLite SQL language server for Claude Code, providing diagnostics, formatting,
completions, and semantic tokens for `.sql` files.

## Supported Extensions
`.sql`

## Installation

The `syntaqlite` binary must be installed and available on your `PATH`.

### Via Homebrew (macOS, recommended)
```bash
brew install LalitMaganti/tap/syntaqlite
```

### Via Cargo (Linux, recommended)
```bash
cargo install syntaqlite-cli
```

### Via download script (all platforms)
```bash
curl -sSf https://raw.githubusercontent.com/LalitMaganti/syntaqlite/main/tools/syntaqlite | python3 - install
```

### Via pip (all platforms)
```bash
pip install syntaqlite
```
